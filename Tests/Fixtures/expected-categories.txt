# Categories emitted by first-party code under subsystem io.goodkind.lmd.
# One per Swift source file that declares an AppLogger category. Categories are
# PascalCase and one-to-one with the Swift type or module they belong to.
#
# This file documents the declared category surface. The
# `swift Tools/lmd-dev.swift log-smoke` command does NOT read this file and does
# NOT assert category coverage; it runs `lmd --help` plus `lmd ls`, captures the
# unified log, and fails only if a line contains a `<private>` redaction. Keep
# this list in sync with the source by hand.
#
# Pure helpers (ANSI escape helpers, layout math, key/mouse byte parsers) do not
# have loggers because they do not fail and have no mutations.
#
# Add new categories here in the same commit that introduces them.
